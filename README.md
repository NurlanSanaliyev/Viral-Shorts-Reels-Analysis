Analysis of YT shorts and Instagram Reels virality (400+ short videos, Kaggle dataset)
Explored whar niches, music types and metrics lead to the most views and engagement.

### Key insights
- Food и Tech — absolute leaders in first-hour views
- Education — the strongest hook (first 3 sec)
- Beauty — the best retention rate
- Original sound boosts views by +30–40% vs trends
- Peak uploading time — sptring and summer (seasonality)
